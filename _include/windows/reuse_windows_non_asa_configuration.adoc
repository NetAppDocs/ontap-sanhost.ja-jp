ASA 以外の設定の場合は、異なる優先順位を持つ 2 つのパスグループが必要です。

優先度が高いパスはアクティブ/最適化されます。つまり、アグリゲートが配置されているコントローラによって処理されます。

優先度の低いパスはアクティブですが、別のコントローラから提供されるため最適化されていません。


NOTE: 最適化されていないパスは、最適化されたパスを使用できない場合にのみ使用されます。
