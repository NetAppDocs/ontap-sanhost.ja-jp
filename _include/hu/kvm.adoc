* Oracle Linux KVMと仮想化について学ぶ
+
Oracle Linux は KVM ホストとして機能できます。これにより、Linux カーネルベースの仮想マシン (KVM) テクノロジを使用して、単一の物理サーバー上で複数の仮想マシンを実行できるようになります。オープンソースの oVirt プロジェクトから構築された Oracle Linux Virtualization Manager を使用すると、複数の Oracle Linux KVM ホストを管理およびサポートできます。KVM ホストでは、 ONTAP LUN に対して明示的なホスト構成設定は必要ありません。
